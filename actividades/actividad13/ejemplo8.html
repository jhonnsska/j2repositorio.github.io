Letter and Word Spacing

by TPS Sena

El kerning es el término que utilizaban los tipógrafos para el espacio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: Letter-spacing. Es particularmente útil para aumentar el kerning cuando su encabezado o frse está en mayúsculas. si el texto está en un caso o normal, aumentar o disminuir el kerning puede dificultar la lectura.

El intervalo predeterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. el intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor de 0,25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
